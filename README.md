## How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/devtraldi/devtraldi_portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd devtraldi_portfolio
   ```
3. Open `index.html` in your web browser.
